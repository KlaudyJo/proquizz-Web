Documentation

GET QUESTIONS

Send a GET request to pl-quizz.com/questions/all to recieve a random array of questions. If limit is not selected, than you will receive 20 random questions in your call. Link: pl-quizz.heroku/questions/all ----> Show in next link.
With Limit: /questions/all ?limit=2 ----> Show in next link.

If you want to select a specific categories or subcategories, than an avaible query parameters are shown below.

GET categories

Send a GET request to pl-quizz.heroku/questions/categories to recieve an information on the categories of question avaible to request. The keys of the returned object are the names for each category. Values of the keys - subcategories, are sent in array when requesting.

GET metadata

Send a GET request to pl-quizz.heroku/questions/metadata to recieve an information on the number of questions in the database and the distribution of questions across categories and subcategories.

Frequently Asked Questions

How do I request questions from API?
It's easy! Just send a HTTP GET request to pl-quizz.com/questions/all and the response will be a JSON array of questions. You don't need an API key.
Can I use your API in my app?
The API and all data returned from it are licensed under a Creative Commons Attribution-NonCommercial 4.0 International License. This means that the API and the questions it returns are free for noncommercial use. If you have somewhere suitable in your app, then please credit the API somewhere discreet - we love having our hard work shared! If you would like to use the API commercially, this service is offered as a paid subscription.
How can I contribute to the project
First of all, you can submit question here. Or if you would like to help with the code and maintaining db, please hget in touch with mail@klaudyjo.cz.
Do I need an api key?
You don't need an API key to request questions. You only need an API key if you want to:
  • Bypass the rate limit
  • Search questions
  • Download all questions
If you would like an API key for any of these reasons, please contact mail@klaudyjo.cz.
Is there a request limit?
For performance reasons we currently limit the amount of questions you can request at one time to 25. We limit the number of requests you can send from a single IP to 20 requests in a 5 second window. If you pass this limit then you will receive HTTP 429 errors. This is to keep the API available to all.
